# Super Get

有前途的有声书下载工具，基于一个大胆的想法。
FastAPI 与 Vue3 架构遥遥领先

## 功能特性

~~反正很牛逼就是了~~

### 搜索

基于热度排序，显示图片、总章节数、艺术家、专辑名称与简介

### 书籍管理

显示添加过的书籍，快速管理任务

显示下载数与总数，帮你直观了解下载进度

支持快速添加定时任务、一键下载全书、移除书籍。

### 章节管理

多选下载、一键全书下载、添加定时任务、查看下载状态

超级流畅的界面，上万章同时显示也不卡顿

### 下载管理

快速开始或取消任务，支持暂停与继续。

独家的定时任务模式，帮你实时获得最新章节

### 设置

前后端分离，支持自主选择后端地址，支持所有设置在网页完成。

独家支持元数据写入，支持多种格式和多种模式。

## 环境要求

- Python 3.8+
- Nodejs

## 安装依赖

```bash
pip install -r requirements.txt
cd web
npm install
```

## 快速开始

1. 安装依赖：
```bash
pip install -r requirements.txt
cd web
npm install
```

2. 运行程序：
```bash
python main.py
```

```bash
cd web

# 编译（于dist）
npm run build
# 预览
npm run preview

# 开发
npm run dev
```
